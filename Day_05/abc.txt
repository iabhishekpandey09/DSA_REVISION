Amazon s3 bucket static website hosting

a) create a s3 bucket
	1. Login to aws console
	2. go to amazon s3
	3. click create bucket
	4. configure
		.bucket name : must be globally unique(my-bucket-name)
		.region : choose the nearest one
		.object ownership : keeps default
		.block public access : uncheck "block all public access", acknowledge warning
	5. create bucket

b)upload static website
	1.open created bucket
	2.click upload -> add files
	3.select all website files
		.index.html
		.style.css
		.script.js
		.images etc
	4.click upload

c) host the static website
	step1 : Enable static website hosting
		1. go to properties -> static website hosting
		2. enable 
			.static website hosting
			.indexed document : index.html
		3. save changes
	step2 : set public access :
		1. go to permission -> bucket policy and add
			{
			   version : "2012-10-17",
			   statement : [
				{
				   "effect" : allow,
				   "principal":"*",
				   "action:"s3:getObject",
				   "resourse" : arn:aws:s3:bucket-name/*"
				}
			     ]
			}


RDP client

a) create an instance
1. login to aws console

3. click launch instance
4. configure
	name : must be globally unique
	awi : microsoft windows server
	instance : t3.micro
	keyapir : select existing pairs or create new pair
5. network setting
	allow RDP(port 3389) required for remote desktop
	optional (allow http/https) if needed
6. click create bucket
7. note the public ip adress

b) connect the window instances using RDP client
	step1 : get administrator password
		1. go to ec2 dashboard
		2. select ur window instance
		3.click connect
		4.go to rdp client tab
		5.click get password
		6.upload your .pem file
		7.click decrypt password
		8.copy :  username : administartor, password : decrypt password

	step2: connecting using rdp

		eneter <public IP>
		click connect
